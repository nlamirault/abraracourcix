# Abraracoursix

[![Travis](https://img.shields.io/travis/nlamirault/abraracoursix.svg)]()

This tool is a simple URL Shortener.
Storage backends supported are :

[x] BoltDB

## Development

* Initialize environment

        $ make init

* Build tool :

        $ make build

* Launch unit tests :

        $ make test
